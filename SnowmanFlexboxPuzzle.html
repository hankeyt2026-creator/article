Flex Properties Used

I used display: flex to create a flex container and flex-wrap: wrap to allow the puzzle pieces to form rows. The puzzle pieces are arranged in the correct order inside the flex container. The incorrect puzzle piece is included in the HTML but hidden using display: none. CSS Grid is used to structure the page into multiple sections for the name, puzzle, and explanation.
